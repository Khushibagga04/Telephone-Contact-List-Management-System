# Telephone-Contact-List-Management-System
**Telephone Contact List Management System** is a simple program that helps users store and manage contacts digitally. It allows adding, deleting, searching, and updating contact details efficiently, providing an easy and organized way to maintain a personal contact list.

FEATURES
- Add new contacts easily
- View all saved contacts
- Delete unwanted contacts
- Search contacts by name
- Update existing contact details
- Fast and efficient data handling

🧠 CONCEPTS USED
- Structures (struct)
- Arrays
- Functions
- Loops (while, for)
- Conditional statements (if, switch)
- String handling (strcmp, scanf)

▶️ HOW TO RUN
- Install a C compiler (like Turbo C / GCC)
- Open the code in any editor (CodeBlocks, VS Code, etc.)
- Compile the program
- Run the program
